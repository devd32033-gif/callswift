# 📞 CallSwift - Telecaller CRM

**CallSwift** is a modern, production-ready telecaller CRM Android app built with Jetpack Compose. It helps telecallers manage leads, log calls, schedule follow-ups, and track performance — with a privacy-first approach.

## Features

- **Lead Management** — Add, edit, search, and filter leads
- **Call Logging** — Auto post-call disposition screen with notes
- **Follow-up Reminders** — Schedule and receive notification reminders
- **Smart Caller ID** — Overlay that shows lead info during calls
- **Disposition Tracking** — Track call outcomes with emoji-based quick selection
- **AI Suggestions** — AI disposition recommendations (Scale plan)
- **Voice Notes** — Record audio notes after calls (Growth+ plan)
- **Admin Dashboard** — Team performance metrics and charts
- **Multi-tenant** — Supports teams with role-based access
- **Offline-first** — Room database with Firebase Firestore sync

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose + Material 3
- **Architecture:** MVVM with Repository pattern
- **Database:** Room (local) + Firebase Firestore (remote)
- **Auth:** Firebase Authentication
- **DI:** Dagger Hilt
- **Notifications:** Firebase Cloud Messaging

## Deployment

This repository hosts the [GitHub Pages](https://pages.github.com/) website for CallSwift. To deploy:
